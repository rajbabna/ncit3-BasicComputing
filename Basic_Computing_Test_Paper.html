Section C
Structured Questions — Answer all parts of each question.
10 marks each  |  20 marks
1.
Word Processing and File Management
A student has been asked to produce a two-page report titled "The Impact of Social Media on Teenagers" using a word processor.
(a) [2 marks] Describe two document format techniques the student should apply to make the report easy to read. For each technique, state what it achieves.
(b) [3 marks] The student notices several spelling errors after typing. Explain the steps she would take to use the spell-check feature, correct the errors, and save the final document with a new file name.
(c) [2 marks] The printer displays an error when she tries to print. Name two possible printer hardware faults and state how each can be remedied.
(d) [3 marks] After completing the report, the student needs to organise her files. Explain three file management techniques she should use to keep her work secure and easy to locate.
2.
Spreadsheet, Internet Research, and Presentation
A student is preparing a class project on the topic "Top 5 Most Visited Countries in the World". She plans to research information online, organise data in a spreadsheet, and present her findings using presentation software.
(a) [3 marks] Describe how the student would use a search engine to find reliable information about this topic. Include in your answer: selecting a search engine, forming a search query using at least one Boolean operator, and evaluating whether the results are relevant.
(b) [4 marks] The student enters her data into a spreadsheet. The table below shows a portion of her data:
A B C D
Country Visitors (millions) Revenue (USD bn) Rank
France 89.4 73.2 1
Spain 83.7 80.0 2
USA 79.3 215.0 3
  1. Write a formula to calculate the total number of visitors (column B, rows 2–6).
  2. Write a formula to find the maximum revenue (column C, rows 2–6).
  3. Describe how the student would produce a bar chart from this data.
  4. State one reason why she should save the file before printing.
(c) [3 marks] The student now creates a presentation of her findings. Describe the four stages of producing and delivering this presentation (design, gather, create, present), giving one specific action for each stage.
