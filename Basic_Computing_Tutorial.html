📝 Competency 1: Operate a Word Processor

Overview

Learn to use word processing software effectively to create, format, and manage documents. This competency covers everything from basic text entry to advanced document manipulation.

Learning Objectives

By the end of this section, you will be able to:

  • Understand basic word processing principles and terminology
  • Use keyboard and mouse commands efficiently
  • Create, format, and manipulate documents
  • Manage printer settings and troubleshoot errors
  • Implement proper file management and security procedures

Basic Principles & Terminology

Word processing is the use of computer software to create, edit, format, and print documents. Key terminology includes:

  • Document: A file containing text and formatting
  • Formatting: The appearance of text (font, size, style)
  • Cursor: The blinking line showing where text will appear
  • Selection: Highlighted text ready for editing
  • Template: A pre-designed document format

Keyboard & Mouse Commands

Mastering keyboard shortcuts improves your productivity significantly:

File Management

Ctrl + N: New document
Ctrl + O: Open
Ctrl + S: Save
Ctrl + P: Print

Text Editing

Ctrl + X: Cut
Ctrl + C: Copy
Ctrl + V: Paste
Ctrl + Z: Undo

Selection

Ctrl + A: Select all
Shift + Arrow: Select text
Double-click: Select word

Document Formatting

Formatting makes your documents visually appealing and professional:

  • Line Formatting: Spacing between lines (single, 1.5, double)
  • Paragraph Formatting: Indentation, alignment (left, center, right, justify), spacing
  • Document Formatting: Page size, margins, headers/footers
  • Font Selection: Font type (Arial, Times New Roman), size (12pt, 14pt), style (bold, italic, underline)
  • Tabulation: Setting tabs for organized alignment of text

Document Manipulation

Essential editing techniques include:

  • Move: Cut text from one location and paste to another
  • Copy: Duplicate text without removing the original
  • Insert: Add new content at cursor position
  • Delete: Remove selected text
  • Find & Replace: Locate specific text and replace it throughout the document
  • Spelling Check: Identify and correct spelling errors
  • Grammar Check: Identify grammatical issues (where available)

💡 Pro Tip

Use Find & Replace (Ctrl + H) to make bulk changes quickly. For example, replacing all instances of "Mr." with "Mr" across your document saves time and ensures consistency.

Screen Display Options

Control your workspace to maximize productivity:

  • Zoom: Adjust view percentage (50%, 100%, 200%) for comfortable reading
  • View Modes: Normal, Page Layout, Outline, Web Layout
  • Ruler: Display/hide rulers for precise measurement
  • Toolbars: Customize which tools are visible
  • Print Preview: See how your document will look when printed

Printer Management

Proper printer use ensures successful document output:

  • Check paper and ink levels before printing
  • Identify common hardware faults (paper jam, low toner)
  • Understand error messages (offline, driver issues)
  • Configure print settings (pages, copies, color/B&W)

File Management & Security

Proper file organization prevents data loss:

  • Save: Use Ctrl + S frequently to preserve changes
  • Save As: Create new versions or change file location
  • Naming: Use descriptive file names (e.g., "Report_2024_Final")
  • Folders: Organize files in logical directory structures
  • Backup: Keep copies of important files
Credits: 5 | Theory Hours: 10 | Practical Hours: 40

Assessment Exercises - Word Processor

1. What is the keyboard shortcut to save a document? ▼

Answer: Ctrl + S (or Cmd + S on Mac)

Explanation: This is the standard save shortcut in most word processors and saves any changes to your document.

2. List three text manipulation techniques you can use in a word processor. ▼

Answer: Any three of:

  • Move (cut and paste)
  • Copy
  • Insert new text
  • Delete
  • Find & Replace
  • Undo/Redo

Explanation: These are essential techniques for editing and modifying documents efficiently.

3. What does the Print Preview function allow you to do? ▼

Answer: Print Preview allows you to see how your document will appear when printed before actually printing it.

Explanation: This helps you identify formatting issues, ensure proper page breaks, and avoid wasting paper on incorrect prints.

4. Multiple Choice: Which of the following is NOT a type of document formatting? ▼
A) Line spacing
B) Font size
C) Document compilation ✓
D) Paragraph alignment

Explanation: Document compilation is not a standard formatting option. The other three are all document formatting features.

5. Practical Exercise: Create and format a document ▼

Task:

  • Open a word processor
  • Type a paragraph about your favorite hobby (minimum 100 words)
  • Apply three different formatting techniques:
    • Change at least two sections to bold
    • Italicize one phrase
    • Change paragraph alignment to center for the title
  • Save the document with a descriptive name
  • Use Find & Replace to change one word throughout the document

Assessment Criteria: Document shows proper use of formatting techniques, file is saved correctly, and find & replace was applied successfully.

📊 Competency 2: Create and Use a Simple Spreadsheet

Overview

Learn to design, create, and use spreadsheets to solve real-world problems. Spreadsheets are powerful tools for organizing data, performing calculations, and visualizing information.

Learning Objectives

By the end of this section, you will be able to:

  • Plan a spreadsheet solution for a given problem
  • Enter and format data effectively
  • Create and use formulas and functions
  • Generate graphs and charts
  • Save and modify spreadsheets

Planning a Spreadsheet

Before creating a spreadsheet, proper planning ensures your solution will be effective:

  • Define the Purpose: What problem are you solving? What question are you answering?
  • Identify Data: What information do you need? Where will it come from?
  • Determine Layout: How will you organize rows and columns?
  • Plan Calculations: What formulas or functions will you need?
  • Design Output: How will you present the results? (tables, charts, reports)

💡 Planning Tip

Sketch your spreadsheet on paper first! Draw the column headers, identify where data will go, and note which cells need formulas. This prevents errors and saves time during creation.

Spreadsheet Components

Understanding the basic structure is essential:

Cells

Individual boxes in the grid where data is stored. Identified by column letter and row number (e.g., A1, B5).

Rows

Horizontal lines of cells numbered 1, 2, 3, etc. Use for data entries.

Columns

Vertical lines of cells labeled A, B, C, etc. Use to categorize information.

Formulas

Mathematical expressions that calculate values based on cell data.

Functions

Pre-built calculations like SUM, AVERAGE, MAX, MIN.

Charts

Visual representations of data (bar, line, pie charts).

Data Entry & Formatting

Proper data entry ensures accuracy and professionalism:

  • Text Entry: Type labels and descriptions in cells
  • Number Entry: Input numerical values for calculations
  • Cell Width: Adjust column width to display content fully
  • Alignment: Left, center, or right align text for clarity
  • Number Format: Display numbers as currency ($), percentages (%), decimals (0.00), or integers
  • Text Format: Apply font styles (bold, italic) to highlight important information

Formulas & Functions

Formulas perform calculations automatically:

  • Basic Arithmetic:
    • Add: =A1+B1
    • Subtract: =A1-B1
    • Multiply: =A1*B1
    • Divide: =A1/B1
  • Common Functions:
    • =SUM(A1:A10) — Adds cells A1 through A10
    • =AVERAGE(A1:A10) — Calculates the average
    • =MAX(A1:A10) — Finds the highest value
    • =MIN(A1:A10) — Finds the lowest value
    • =COUNT(A1:A10) — Counts number entries

Creating Graphs & Charts

Visualizations make data easier to understand:

  • Step 1: Select the data range you want to visualize
  • Step 2: Insert a chart (usually via Insert menu)
  • Step 3: Choose chart type:
    • Bar Chart: Compare categories
    • Line Chart: Show trends over time
    • Pie Chart: Show proportions/percentages
    • Column Chart: Similar to bar, with vertical columns
  • Step 4: Add title and labels for clarity

Saving & Using Spreadsheets

Manage your spreadsheets effectively:

  • Save: Use Ctrl + S to save your work frequently
  • File Name: Use descriptive names (e.g., "Sales_Q4_2024")
  • File Format: Save as .xlsx (Excel) for compatibility
  • Modify: Open saved files, change variables, and recalculate automatically
  • Print: Print as hard copy or export as PDF
Credits: 5 | Theory Hours: 10 | Practical Hours: 40

Assessment Exercises - Spreadsheet

1. What is the purpose of planning before creating a spreadsheet? ▼

Answer: Planning ensures you:

  • Understand the problem you're solving
  • Organize data logically
  • Include necessary formulas and calculations
  • Create an effective solution efficiently

Explanation: Good planning prevents mistakes and saves time during creation.

2. Write a formula to calculate the average of cells B1 through B10. ▼

Answer: =AVERAGE(B1:B10)

Explanation: The AVERAGE function calculates the mean of the specified range. The colon (:) indicates a range from B1 to B10 inclusive.

3. What are two ways to apply a formula to multiple cells? ▼

Answer: Two common methods:

  • Fill Down: Enter formula in first cell, select range, use Ctrl + D
  • Drag Formula: Click cell with formula, drag the small square in bottom-right corner down to desired cells

Explanation: Both methods copy the formula and adjust cell references automatically.

4. Multiple Choice: Which chart type is best for showing a trend over time? ▼
A) Pie Chart
B) Line Chart ✓
C) Donut Chart
D) Scatter Plot

Explanation: Line charts are ideal for showing trends, changes, and progression over time because they clearly display the direction and rate of change.

5. Practical Exercise: Create a student grade spreadsheet ▼

Task:

  1. Create a spreadsheet with the following structure:
    • Column A: Student names (minimum 5 students)
    • Column B: Math test scores
    • Column C: English test scores
    • Column D: Science test scores
  2. Add formulas in Column E to calculate the average for each student
  3. Add a summary row below data showing:
    • Maximum score for each subject
    • Average score for each subject
  4. Format the spreadsheet:
    • Bold the header row
    • Display scores with one decimal place
  5. Create a bar chart comparing average scores by subject
  6. Save the file as "Student_Grades"

Assessment Criteria: Correct data entry, proper formulas, accurate calculations, professional formatting, and functional chart.

🎤 Competency 3: Use Computer Technology for Presentations

Overview

Learn to design, create, and deliver professional presentations using computer software. Presentations are essential for communicating information effectively to audiences.

Learning Objectives

By the end of this section, you will be able to:

  • Design effective presentations for specific audiences
  • Gather and organize relevant information
  • Create visually appealing slides
  • Deliver presentations confidently
  • Use presentation software features effectively

Planning Your Presentation

Successful presentations start with careful planning:

  • Topic: Choose a clear, focused subject
  • Purpose: What do you want the audience to learn or do?
  • Target Audience: Who are you presenting to? (students, colleagues, clients)
  • Method: How will you present? (in-person, online, hybrid)
  • Duration: How much time do you have?
  • Resources: What software and tools will you use?

💡 Planning Tip

Use the "Rule of Three": Organize your presentation into three main points or sections. This helps audiences remember your key messages and makes structure clear.

Gathering & Organizing Information

Quality content is crucial for effective presentations:

  • Research: Find reliable sources (books, websites, academic journals)
  • Relevance: Ensure all information relates to your topic and audience
  • Accuracy: Verify facts and data are correct
  • Organization: Arrange information logically (chronological, problem-solution, cause-effect)
  • Summary: Provide clear takeaways and conclusions

Presentation Design

Visual design significantly impacts audience engagement:

Layout & Structure

Use consistent templates. Keep slides uncluttered with adequate white space for readability.

Color Scheme

Choose 2-3 complementary colors. Ensure high contrast between text and background.

Typography

Use readable fonts (sans-serif preferred). Maintain consistent sizing: titles larger than body text.

Visual Elements

Include relevant images, charts, and diagrams. Avoid excessive animations that distract.

Content Balance

Follow the "6x6 rule": maximum 6 bullets per slide, 6 words per bullet.

Consistency

Maintain consistent styling across all slides for a professional appearance.

Creating Slides

Best practices for slide creation:

  • Title Slide: Include presentation title, your name, date, and institution
  • Content Slides: One main idea per slide, with supporting details
  • Bullet Points: Use short phrases, not full sentences
  • Visual Hierarchy: Make important information more prominent
  • Transitions: Use subtle transitions between slides (avoid excessive effects)
  • Speaker Notes: Include detailed notes for reference during presentation

Presentation Software

Common tools and their features:

  • Microsoft PowerPoint: Industry standard with extensive formatting options
  • Google Slides: Cloud-based, collaborative, accessible from any device
  • Apple Keynote: Mac-based with elegant design templates
  • LibreOffice Impress: Free, open-source alternative

Delivering Your Presentation

Effective delivery enhances your message:

  • Practice: Rehearse multiple times to build confidence
  • Timing: Know your pacing; don't rush or speak too slowly
  • Eye Contact: Look at different audience members
  • Body Language: Stand confidently, use natural gestures
  • Voice: Speak clearly, vary tone to maintain interest
  • Interaction: Engage audience with questions or discussions
  • Backup Plan: Have printed slides in case of technical issues
Credits: 5 | Theory Hours: 10 | Practical Hours: 40

Assessment Exercises - Presentation

1. What are the three key planning elements before creating a presentation? ▼

Answer: The three key elements are:

  1. Topic: The subject matter
  2. Purpose: What you want the audience to learn
  3. Target Audience: Who you're presenting to

Explanation: These elements guide all other decisions about content, design, and delivery.

2. What is the "6x6 Rule" in presentation design? ▼

Answer: Maximum 6 bullet points per slide with maximum 6 words per bullet point.

Explanation: This rule prevents slides from becoming cluttered and ensures the audience isn't overwhelmed with text. It encourages concise messaging and keeps focus on your verbal delivery.

3. Name three key elements to consider when checking information for relevance. ▼

Answer: Three elements to consider:

  • Topic Relevance: Does the information relate to your topic?
  • Purpose Alignment: Does it support your presentation's purpose?
  • Audience Appropriateness: Is it suitable and interesting for your target audience?

Explanation: Relevant information keeps your presentation focused and engaging for your specific audience.

4. Multiple Choice: Which is NOT recommended for effective slide design? ▼
A) Consistent color scheme
B) High contrast between text and background
C) Filling slides with as much text as possible ✓
D) Using readable fonts

Explanation: Crowded slides with excessive text are hard to read, distract from your presentation, and violate the 6x6 rule. Less text is better.

5. Practical Exercise: Create a presentation on a computing topic ▼

Task:

  1. Choose a computing topic (e.g., "Cybersecurity Basics," "Artificial Intelligence," "Cloud Computing")
  2. Plan your presentation:
    • Define topic, purpose, and target audience
    • Outline main points (3-5 sections)
  3. Create a presentation with minimum 8 slides:
    • Slide 1: Title slide with topic, name, date
    • Slides 2-6: Content slides (one point each)
    • Slide 7: Summary/Key Takeaways
    • Slide 8: Q&A or Conclusion
  4. Apply consistent design:
    • Use professional color scheme (2-3 colors)
    • Readable fonts (title and body fonts)
    • Include at least 2 relevant images or charts
  5. Add speaker notes to each content slide
  6. Practice delivering the presentation (time yourself)

Assessment Criteria: Clear topic and purpose, relevant information, professional design, proper use of software features, and effective delivery.

🌐 Competency 4: Use a Browser and Email

Overview

Master internet navigation and email communication. These skills are essential for research, information gathering, and digital communication in today's world.

Learning Objectives

By the end of this section, you will be able to:

  • Launch and navigate web browsers effectively
  • Use search engines with basic and advanced techniques
  • Understand and use hyperlinks
  • Send and receive emails with attachments
  • Configure email settings

Web Browsers

A web browser is software that displays web pages from the internet. Popular browsers include:

Google Chrome

Fast, user-friendly, syncs across devices, strong security.

Mozilla Firefox

Open-source, privacy-focused, excellent customization.

Safari

Mac/iOS native browser, fast, good battery performance.

Microsoft Edge

Modern browser, integrates with Windows, fast performance.

Opera

Lightweight, built-in VPN, good for slower connections.

Brave

Privacy-focused, blocks ads and trackers by default.

Browser Navigation

Essential browser controls:

  • Address Bar: Type URL (e.g., www.google.com) to visit websites
  • Back Button: Return to previously visited pages
  • Forward Button: Move forward after using back button
  • Home Button: Return to your browser's home page
  • Refresh/Reload: Reload the current page (F5 or Ctrl+R)
  • Stop Button: Stop loading a page
  • Bookmarks/Favorites: Save frequently visited websites for quick access
  • History: View previously visited websites
  • Search Bar: Search directly from the browser

URL Structure

Understanding URLs helps you navigate effectively:

  • Protocol: https:// (secure) or http:// (less secure)
  • Domain: The website name (example.com)
  • Path: Specific page or resource (/products/item)
  • Example: https://www.example.com/products/item

Search Engines

Search engines help you find information online. Popular options include:

  • Google: Largest search engine, comprehensive results
  • Bing: Microsoft search engine, integration with Windows
  • Yahoo: Directory-based search, news integration
  • DuckDuckGo: Privacy-focused, no tracking

Search Techniques

Effective searching saves time and improves results:

  • Simple Search: Single word or short phrase
    • Example: "healthy recipes"
  • Phrase Search: Put exact phrase in quotes for exact matches
    • Example: "climate change impact"
  • Boolean Operators: Use AND, OR, NOT to refine searches
    • AND: "machine learning AND Python"
    • OR: "Python OR Java" (either term)
    • NOT: "smartphones NOT iPhones"
  • Advanced Search: Most search engines offer advanced search options
    • Search by date range
    • Search specific websites
    • Search by file type (PDF, DOC)

Hyperlinks

Hyperlinks connect different web pages and resources:

  • Definition: Clickable elements linking to other content
  • Text Links: Highlighted text (usually blue and underlined)
  • Image Links: Clickable images
  • Email Links: Links that open your email client
  • Types:
    • Internal links: Navigate within the same website
    • External links: Navigate to different websites

Email Fundamentals

Email is essential for digital communication:

  • Email Client: Software for sending/receiving emails (Outlook, Gmail, Thunderbird)
  • Email Address: Unique identifier (username@domain.com)
  • Inbox: Folder receiving incoming messages
  • Outbox: Folder holding messages before sending
  • Sent Items: Archive of messages you've sent

Composing & Sending Emails

Best practices for effective email communication:

  • To: Primary recipient's email address
  • Cc: Carbon copy (recipient can see others copied)
  • Bcc: Blind carbon copy (hidden from other recipients)
  • Subject: Clear, descriptive subject line
  • Body: Professional message with proper greeting and closing
  • Attachments: Add files by clicking attachment icon
  • Signature: Auto-add your contact information

💡 Email Etiquette

Always use a professional tone, proofread before sending, respond within 24 hours, and use appropriate formality for your relationship with the recipient.

Email Features

Useful email configuration options:

  • Signature File: Automatic footer added to all emails
  • Read Receipts: Confirm when recipient opens your email
  • Auto-Reply: Automatic response when away
  • Folders/Labels: Organize emails by topic or sender
  • Filters/Rules: Automatically sort incoming mail
  • Contacts/Address Book: Store frequently used email addresses

Assessment Exercises - Web & Email

1. Define a web browser and list its main purpose. ▼

Answer: A web browser is software that displays web pages from the internet. Its main purpose is to access, view, and navigate information on the World Wide Web.

Explanation: Browsers interpret HTML code and render it as viewable pages that users can interact with.

2. Explain the difference between AND, OR, and NOT Boolean operators in searching. ▼

Answer:

  • AND: Results must contain both terms (narrows search)
  • OR: Results contain either term (expands search)
  • NOT: Excludes a specific term from results (narrows search)

Examples:

  • "Python AND programming" - Results contain both words
  • "Java OR JavaScript" - Results contain either language
  • "programming NOT games" - Results exclude gaming references
3. List three browser controls and their functions. ▼

Answer: Any three of:

  • Back Button: Returns to previously visited page
  • Forward Button: Goes to next page after using back
  • Refresh Button: Reloads current page
  • Home Button: Returns to browser's home page
  • Bookmarks: Saves frequently visited sites
  • History: Shows previously visited pages

Explanation: These controls make navigation efficient and organized.

4. Multiple Choice: What is the purpose of the BCC (Blind Carbon Copy) field in email? ▼
A) To send copies to recipients who can see other recipients
B) To send copies to recipients without revealing them to other recipients ✓
C) To mark email as important
D) To create a backup of the email

Explanation: BCC keeps recipient addresses hidden from other recipients, useful for privacy and mass communications.
