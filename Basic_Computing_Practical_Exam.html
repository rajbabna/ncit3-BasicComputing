Question 1: Word Processing & Document Creation
25 Marks
Competency: Operate a Word Processor
Scenario

You are a secretary at a small company. You have been asked to create a professional business letter and a simple informational document.

Task

Using a word processor, create a professional business letter with the following specifications:

1. Letter Format 5 marks
  • Include proper letter layout with sender's address, date, recipient's address, and salutation
  • Use appropriate spacing and margins
  • Include a closing and signature line
2. Content and Formatting 8 marks
  • Write a letter (minimum 200 words) requesting information about a computer training program
  • Apply at least THREE different formatting techniques:
    • Bold text for important points
    • Bullet points or numbered lists for key information
    • Italics or underline for emphasis
  • Ensure proper paragraph formatting with adequate spacing between paragraphs
3. Document Manipulation 7 marks
  • Use Find and Replace to change all instances of a word (e.g., replace "course" with "training program")
  • Demonstrate ability to copy and paste a section within the document
  • Show evidence of spell checking and grammar checking
  • Include a footer with the document name and page number
4. File Management and Printing 5 marks
  • Save the document with an appropriate filename
  • Display the Print Preview
  • Show the directory/folder contents demonstrating file location
Marking Breakdown
  • Letter format and structure5 marks
  • Content quality and formatting techniques8 marks
  • Document manipulation and editing7 marks
  • File management and output5 marks
  • Total25 marks
Question 2: Spreadsheet Problem-Solving
25 Marks
Competency: Create and Use a Simple Computer Spreadsheet to Solve a Problem
Scenario

A small retail store owner needs to track monthly sales for three product categories (Electronics, Clothing, Home & Garden) over a three-month period (January, February, March). The owner needs to calculate totals, averages, and create a visual representation of the data.

Task

Create a spreadsheet to solve this problem with the following requirements:

1. Data Entry and Planning 4 marks
  • Create a working plan documenting:
    • Purpose of the spreadsheet (sales tracking and analysis)
    • Spreadsheet specifications required
  • Enter sample sales data for the three categories across three months
  • Organize data in a logical table format with clear headers
2. Formatting and Cell Design 5 marks
  • Format column headers (bold, centered, colored background)
  • Adjust cell widths appropriately for readability
  • Apply number formatting to currency values (show as $ with 2 decimal places)
  • Apply text alignment (left, center, right) where appropriate
  • Use borders to define table structure
3. Formulas and Functions 8 marks
  • Calculate the monthly total sales (use SUM function) – 2 marks
  • Calculate the average sales per category (use AVERAGE function) – 2 marks
  • Calculate the maximum sales value (use MAX function) – 2 marks
  • Create at least one additional calculation (e.g., total sales across all categories, percentage of total) – 2 marks
4. Data Visualization and Output 5 marks
  • Create a graph/chart using the data (bar chart, column chart, or pie chart) – 3 marks
  • Save the spreadsheet file with appropriate naming – 1 mark
  • Print or export the spreadsheet in PDF format showing both data and graph – 1 mark
5. Problem Solution and Modification 3 marks
  • Open the saved spreadsheet file
  • Modify one data entry and demonstrate that formulas update automatically
  • Show ability to use cell ranges for creating the graph
Marking Breakdown
  • Data entry and planning4 marks
  • Formatting and presentation5 marks
  • Formulas and functions8 marks
  • Data visualization and output5 marks
  • Problem solution and file manipulation3 marks
  • Total25 marks
Question 3: Presentation Design & Creation
25 Marks
Competency: Use Computer Technology for Presentation Purposes
Scenario

You have been tasked to create a presentation introducing "Cybersecurity Basics" to new employees in your organization. The presentation should be educational, professional, and engaging.

Task

Design and create a presentation with the following specifications:

1. Presentation Design and Planning 6 marks
  • Document the presentation design including:
    • Topic: Cybersecurity Basics – 1 mark
    • Purpose: Educate new employees on essential cybersecurity practices – 1 mark
    • Target Audience: New employees with varying IT knowledge – 1 mark
    • Method of Presentation: Digital slide presentation using presentation software – 1 mark
    • List 3-4 credible sources of information for the topic – 2 marks
  • Outline the presentation structure and content plan (main topics to be covered)
2. Information Gathering and Content Organization 5 marks
  • Gather information on cybersecurity basics covering:
    • Password security
    • Phishing awareness
    • Data protection
    • Safe browsing practices
  • Ensure information is relevant to the audience and purpose
  • Organize information logically with clear sections/slides
3. Presentation Creation 8 marks
  • Create a minimum of 6 slides:
    • Title slide with topic, presenter name, and date – 1 mark
    • 4 content slides covering the cybersecurity topics – 4 marks
    • Conclusion/Summary slide – 1 mark
  • Ensure consistent design throughout (colors, fonts, layout)
  • Include at least one image, diagram, or visual element per content slide
  • Use appropriate text formatting and font sizes for readability
  • Keep text concise with bullet points rather than paragraphs
4. Editing and Refinement 3 marks
  • Review content for consistency with design plan
  • Edit information for clarity and accuracy
  • Ensure formatting aligns with presentation design
  • Check for spelling and grammar errors
5. Presentation Delivery 3 marks
  • Prepare to present the topic to the audience (new employees)
  • Communicate information clearly using appropriate language
  • Use the computer application effectively to display slides
  • Demonstrate understanding of the topic content
Marking Breakdown
  • Design planning and documentation6 marks
  • Information gathering and organization5 marks
  • Presentation creation and design8 marks
  • Editing and refinement3 marks
  • Presentation delivery3 marks
  • Total25 marks
